# Rijoma — Produção Portuguesa de Malhas

Site institucional da **Rijoma, Lda**, fabricante português de malhas retilíneas desde 1986, em Minde.

## Estrutura

```
index.html    — Página única (hero, sobre, produção, produtos, personalização, contactos)
styles.css    — Estilos responsivos (desktop, tablet, mobile)
assets/       — Imagens e logótipos
```

## Desenvolvimento

Abrir `index.html` num browser — não há dependências nem build step.
